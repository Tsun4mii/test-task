## Installation

```bash
$ npm install
```

### Fill .env file

```bash
DATABASE_URL="postgresql://{db_user}:{db_pass}@{db_host}:5432/{db_name}"
```

## Running the app

```bash
$ npm run start:deploy
```

## Documentation

Documentation can be obtained from the following URL:

```bash
http://localhost:3000/api#
```
